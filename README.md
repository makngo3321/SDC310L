# Project Name: Makayla's Store

## Project Description: This is a storefront webpage to order groceries. It provides users with a seamless experience to browse the grocery list, add to their cart, and checkout. 

## Project Tasks:
- Started with adding information into PHPMyAdmin - Such as creating the database and tables.
  - Followed-up by adding in attributes such as the items in my grocery store.
- CRUD Operations were next. I started to create more of the functionality and more pages.
  - This includes selections to view your cart, continue shopping, checkout and more.
  - I also added in all aspects of code at this stage to create the webpage, and take it less from just data to a real website.
- MVC Framework: I added in the framework aspects we learned in the main course, making sure all functionality for this aspect worked well.
- Design: I created a styles.css page to design the website - focusing on greens and changing up the font style to be more approachable and easier on the eyes.
- Debug: I then began to debug - I ran into issues that were easy to fix once discovered.
- Testing: Completed a full review, documented my findings and how I tested the application in addition to correcting any bugs, and created a README file.

## Project Skills Learned:
- Frontend development, such as CSS and HTML
- Github Tags and Releases, along with updates
- Server-Side Scripting with PHP
- Writing documentation and README files
